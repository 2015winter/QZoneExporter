# QQ空间导出助手 - 常见问题

> **FAQ for QZone Exporter**

---

## 🔍 问题排查指南

遇到问题时，请按以下顺序自行排查：

```
1. 确认 QQ 空间是否正常访问
2. 刷新 QQ 空间页面
3. 重启浏览器
4. 检查网络连接
5. 重新安装助手
6. 重启电脑
```

> 💡 **经验法则：** 关机重启、卸载重装可解决 99% 的问题

---

## 🔐 账号安全

### Q: 使用助手安全吗？

**是安全的。**

| 安全保障 | 说明 |
|---------|------|
| 开源代码 | 源码公开可审查 |
| 本地处理 | 数据不上传服务器 |
| 无后门 | 官方版无恶意代码 |

> ⚠️ **警告：** 第三方修改版不保证安全，请从官方渠道获取

### Q: 会导致 QQ 号被冻结/封号吗？

| 情况 | 说明 | 解决方案 |
|------|------|---------|
| **临时冻结** | 采集速度过快可能触发 | 调大查询间隔，等待自动解冻 |
| **封号** | 理论上不会，但不绝对保证 | 如介意风险请勿使用 |

### Q: 隐私会泄露吗？

**助手承诺不泄露隐私。**

但以下情况助手不负责：
- 使用第三方修改版
- 备份文件丢失/泄露（如 U 盘丢失、上传公网）
- 导出他人数据并传播

## 📦 备份相关

### Q: 如何备份别人的 QQ 空间？

1. 登录自己的 QQ 账号
2. 访问他人空间
3. 在他人空间页面点击助手图标开始备份

**注意：**
- 会留下访客记录（黄钻除外）
- 只能备份有权限访问的内容

### Q: 能备份加密空间/相册吗？

| 情况 | 是否可备份 |
|------|-----------|
| 对你开放权限 | ✅ 可以 |
| 正确回答密码问题 | ✅ 可以 |
| 无访问权限 | ❌ 不可以 |

### Q: QQ 空间被封能备份吗？

**部分可以。**

| 模块 | 是否支持 |
|------|---------|
| 说说、视频、留言、收藏、访客、好友 | ✅ 支持 |
| 日志、日记、相册、分享 | ❌ 不支持 |

### Q: 能备份私密日记吗？

**可以，但需要：**

1. 进入日记设置
2. **关闭**「独立密码认证」
3. 完成备份后可重新开启

### Q: 有手机版/APP 吗？

**没有。** 助手仅有浏览器扩展版本。

移动端使用需要：
- 支持扩展的浏览器（如 Kiwi Browser）
- 可能存在兼容性问题

## 🖱️ 使用问题

### Q: 点击图标无反应？

**检查项：**

| 检查项 | 正确状态 |
|--------|---------|
| 登录状态 | 已登录 QQ 空间 |
| 页面地址 | 以 `user.qzone.qq.com/QQ号` 开头 |
| 扩展状态 | 已启用 |

### Q: 采集过程中数据被跳过？

**正常现象，无需担心。**

跳过原因：
- 非长说说 → 跳过全文获取
- 图片 ≤9 张 → 跳过更多图片
- 无语音 → 跳过语音获取
- 检测自己 → 跳过互动信息

### Q: 备份文件在哪里？

| 文件类型 | 位置 |
|---------|------|
| 文案压缩包 | 浏览器默认下载目录 |
| 图片/视频 | 下载工具设置的目录 |

### Q: 数据缺少或丢失？

| 原因 | 解决方案 |
|------|---------|
| 采集频率过快 | 调大「查询间隔」重新备份 |
| QQ 空间已缺失 | 助手无法恢复已删除数据 |
| 网络不稳定 | 增加「列表重试」次数 |

## 🖼️ 相册相关

### Q: 下载的是原图吗？

**默认是原图。**

| 情况 | 说明 |
|------|------|
| 相册图片 | 可在配置中指定「相片质量」 |
| 说说配图 | 通常没有原图，除非引用相册 |

如原图不存在，会按质量递减下载。

### Q: Exif 信息丢失？

取决于 QQ 空间原图是否保留了 Exif 信息。助手不会额外删除 Exif。

### Q: 相片创建时间不对？

- **文件时间：** 默认为下载时间
- **拍摄时间：** 保存在 Exif 信息中

## 📺 查看备份

### Q: 如何查看 HTML 备份？

双击 `index.html` 文件，在浏览器中打开。

### Q: 如何查看 Markdown 备份？

使用 Markdown 编辑器打开：
- Typora
- VS Code
- Mark Text

### Q: 页面显示异常？

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 图片不显示 | 文件夹未合并 | 合并文案和多媒体文件夹 |
| 图片不显示 | 图片下载失败 | 检查下载任务，重新下载 |
| 样式错乱 | CDN 加载失败 | 切换为离线 CDN |

### Q: 支持离线查看吗？

**支持，但需配置。**

详见 [离线查看文档](./offline.md)

> ⚠️ **注意：** 部分 QQ 官方在线表情离线后仍无法显示

## ⬇️ 下载工具

### Q: 浏览器下载不停弹窗？

**解决方案：**

```
浏览器设置 → 下载 → 关闭「下载前询问每个文件的保存位置」
```

### Q: 浏览器下载提示无权限？

| 可能原因 | 解决方案 |
|---------|---------|
| 浏览器限制 | 更换 Motrix/Aria2 下载 |
| 域名限制 | 在配置中添加引用来源域名 |

### Q: 迅雷无法唤起？

**检查项：**

1. 是否安装官方正版迅雷（v10+）
2. 是否被安全软件拦截
3. 尝试使用「迅雷下载链接」模式

### Q: Aria2/Motrix 无法添加任务？

**检查项：**

| 检查项 | 正确配置 |
|--------|---------|
| 软件版本 | v1.35.0 以上 |
| RPC 服务 | 已开启 |
| RPC 地址 | 正确填写（Motrix: 16800, Aria2: 6800） |
| RPC 密钥 | 与软件设置一致 |

## ⚙️ 配置问题

### Q: 配置页面显示错乱？

| 可能原因 | 解决方案 |
|---------|---------|
| 网络不稳定 | 刷新页面 |
| CDN 加载失败 | 切换离线 CDN |
| 版本过旧 | 更新助手版本 |

### Q: 配置保存后不生效？

**必须刷新 QQ 空间页面！**

配置保存后，需要：
1. 刷新 QQ 空间页面
2. 或关闭后重新打开 QQ 空间

## 🚫 说说限制

### Q: 提示「使用人数过多，请稍后重试」？

**原因：** 采集频率过快

**解决方案：**

1. 调整「查询间隔」（建议 10-30 秒）
2. 在「稍候重试接口」中勾选「说说列表」
3. 设置重试次数和间隔

助手会自动等待重试，直到次数用尽跳过。

**没有。** 本项目仅支持 QQ 空间。

基于 QQ 空间接口检测，准确率较高但不保证 100%。
