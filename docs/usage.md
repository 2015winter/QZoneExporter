# QQ空间导出助手 - 使用指南

> **Usage Guide for QZone Exporter**

---

## 📚 术语说明

在开始之前，请先了解以下术语：

| 术语 | 说明 |
|------|------|
| **文案内容** | 说说、日志、评论等文字内容（HTML 格式） |
| **多媒体文件** | 图片、视频、语音等二进制文件 |
| **备份压缩包** | 采集完成后打包下载的 ZIP 文件，包含文案内容 |
| **完整备份** | 文案内容 + 多媒体文件合并后的完整数据 |

└─────────────┘    └─────────────┘    └─────────────┘

## 📋 详细步骤

### 步骤 1：配置助手

1. 右键点击助手图标 → 选择「选项」
2. 根据需要调整配置（详见 [配置说明](./configuration.md)）
3. 点击「保存」

**推荐配置：**
- 下载工具：Motrix（需提前安装）
- 备份类型：HTML
- 查询间隔：3-5 秒

### 步骤 2：访问 QQ 空间

1. 打开浏览器，访问 [QQ空间](https://qzone.qq.com)
2. 登录你的 QQ 账号
3. 进入需要备份的空间：
   - **备份自己：** 点击「我的主页」
   - **备份他人：** 搜索或访问他人空间

> ⚠️ **重要：** 必须在 QQ 空间页面（URL 以 `user.qzone.qq.com` 开头）才能使用助手

### 步骤 3：开始备份

1. 点击浏览器工具栏的助手图标
2. 选择需要备份的内容模块：

| 模块 | 说明 |
|------|------|
| ☐ 说说 | 包含文字、图片、语音、视频说说 |
| ☐ 日志 | 公开日志 |
| ☐ 日记 | 私密日记（需关闭独立密码） |
| ☐ 相册 | 相册及相片 |
| ☐ 视频 | QQ空间视频 |
| ☐ 留言 | 留言板内容 |
| ☐ 好友 | QQ好友列表 |
| ☐ 收藏 | 收藏夹内容 |
| ☐ 分享 | 分享的内容 |
| ☐ 访客 | 空间访客记录 |

3. 点击「开始备份」

### 步骤 4：等待采集完成

- 采集过程中会显示进度提示
- **请勿关闭、刷新或打开新的 QQ 空间页面**
- 采集时长取决于数据量，主号建议睡前备份

### 步骤 5：下载备份文件

采集完成后：

1. **下载文案压缩包：** 点击「打包下载」按钮
   - 下载 `QQ空间备份_QQ号.zip` 文件
   - 解压得到 `QQ空间备份_QQ号` 文件夹

2. **等待多媒体下载：** 
   - 助手会自动将图片/视频添加到下载工具
   - 等待所有下载任务完成
   - 文件位于下载工具设置的目录中

### 步骤 6：合并文件夹

将两个文件夹的内容合并：

```
文案内容文件夹（解压得到）        多媒体文件夹（下载得到）
QQ空间备份_123456/               QQ空间备份_123456/
├── index.html                   ├── Albums/
├── Common/                      │   └── [图片文件]
├── Messages/                    ├── Messages/
│   └── index.html               │   └── [图片文件]
└── ...                          └── Videos/
                                     └── [视频文件]
```

**合并方法：** 将其中一个文件夹的所有内容复制/移动到另一个文件夹中

### 步骤 7：查看备份

1. 打开合并后的文件夹
2. 双击 `index.html` 文件
3. 在浏览器中查看备份内容

## 🎯 功能详解

### 打包下载

- **出现时机：** 大多数备份完成后都会显示
- **包含内容：** 文案内容（HTML/MD/JSON 文件）
- **不包含：** 图片、视频等多媒体文件

**例外情况：** 仅备份相册/视频且类型为「文件」时，不显示打包按钮（直接下载文件）

### 下载管理

点击「下载管理」可以：

- 查看下载任务状态
- 重试失败的任务
- 切换下载工具重新下载

### 查看备份

点击「查看备份」可以打开备份文件夹位置。

## 🔄 增量备份

### 什么是增量备份

- **全量备份：** 备份所有内容
- **增量备份：** 只备份新增/更新的内容

### 使用场景

1. 首次全量备份后，定期进行增量备份
2. 避免重复下载已有内容
3. 节省时间和空间

### 使用方法

1. **首次备份：** 选择「全量」模式，得到文件夹 A
2. **后续备份：** 选择「上次备份」模式，得到文件夹 B
3. **合并结果：** 将文件夹 B 的内容合并到文件夹 A

### 注意事项

| 情况 | 说明 |
|------|------|
| 已删除内容 | 增量备份不会同步删除（备份中仍保留） |
| 旧内容新评论 | 不会备份旧内容的新增评论 |
| 时间计算 | 基于内容发表时间，非服务器时间 |

## 💾 备份管理

### 功能入口

选项 → 工具 → 备份管理

### 功能说明

| 功能 | 说明 |
|------|------|
| **导入** | 将本地备份数据导入浏览器 |
| **导出** | 将浏览器中的备份记录导出保存 |
| **删除** | 清除浏览器中的备份记录 |

### 使用建议

- 定期导出备份记录，防止浏览器清理导致丢失
- 更换浏览器/电脑时，导出后在新环境导入
- 删除记录后，下次备份将变为全量备份

## 📷 本地相册功能

### 功能说明

将本地电脑中的照片整合到备份的相册展示页面中。

### 前提条件

- 相册模块备份类型必须为 HTML
- 已完成至少一次完整备份

### 使用步骤

1. 在完整备份文件夹的 `Albums` 目录下新建分类文件夹
   ```
   Albums/
   ├── 原有相册/
   └── 本地相册/        ← 新建
       └── 旅行照片/    ← 放入本地照片
   ```

2. 打开助手选项 → 工具 → 本地相册

3. 选择备份文件夹，点击生成

4. 刷新 HTML 即可在相册中查看本地照片

## 📝 备份建议

### 时间选择

| 建议 | 原因 |
|------|------|
| **睡前备份** | 无人值守，不影响使用 |
| **网络空闲时** | 避免网络拥堵 |
| **工作日备份** | 服务器相对空闲 |

### 分批备份

数据量过大时，可以分模块备份：

1. 第一次：说说 + 日志 + 日记
2. 第二次：相册 + 视频
3. 第三次：其他模块
4. 最后合并所有文件夹

### 多重备份

| 存储位置 | 用途 |
|---------|------|
| 本地硬盘 | 主备份 |
| 移动硬盘 | 离线备份 |
| 网盘 | 云端备份（注意隐私） |

## ⚠️ 常见问题

### Q: 点击图标无反应？

**检查项：**
1. 是否已登录 QQ 空间
2. 地址栏是否以 `user.qzone.qq.com` 开头
3. 尝试刷新页面

### Q: 采集过程中数据被跳过？

**正常情况，可能原因：**
- 非长说说跳过全文获取
- 图片未超过 9 张跳过更多图片
- 无语音内容跳过语音获取
- 自己的好友跳过互动检测

### Q: 备份文件在哪里？

| 内容 | 位置 |
|------|------|
| 文案压缩包 | 浏览器默认下载目录 |
| 多媒体文件 | 下载工具设置的目录 |

### Q: 页面显示异常？

**可能原因：**
1. 文件夹未正确合并
2. 图片下载失败或原图已失效
3. CDN 加载失败（建议切换离线模式）

更多问题请参考 [常见问题](./faq.md)
